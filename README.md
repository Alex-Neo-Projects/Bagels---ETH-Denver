## Installation 
Download [bagels](https://github.com/Alex-Neo-Projects/bagels) via npm
```
npm i -g bagels
```

Clone this repository: 
```
git clone https://github.com/Alex-Neo-Projects/Bagels---ETH-Denver.git
```

``cd`` into the ```contracts``` folder:
```
cd contracts
```

Run Bagels: 
```
bagels
```
